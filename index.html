2.부트스트랩 - 레이아웃

2.1 중단점 (breakpoints)

■ 정의

웹사이트 또는 웹 애플리케이션의 레이아웃을 반응형으로 만들기 위한 디자인 요소입니다.
미디어 쿼리를 사용하여 설정되며, 특정 디바이스 크기에 따라 레이아웃이 변경될 수 있도록 합니다.

■ 6개의 브레이크포인트

xs (Extra small)
<576px
가장 작은 화면 크기를 위한 브레이크포인트로, 모바일 기기 등의 매우 작은 화면 크기를 대상으로 합니다.

sm (Small)
≥576px
작은 화면 크기를 대응하기 위한 브레이크포인트로, 태블릿과 같은 중간 정도의 화면 크기를 포함합니다.

md (Medium)
≥768px
설명: 중간 크기의 데스크탑 화면을 대응하기 위한 브레이크포인트입니다.

lg (Large)
≥992px
큰 데스크탑 화면을 위한 브레이크포인트로, 일반적인 데스크탑 화면 크기를 대상으로 합니다.

xl (Extra large)
≥1200px
매우 큰 데스크탑 화면을 위한 브레이크포인트로, 대형 모니터 등을 대상으로 합니다.

xxl (Extra extra large)
≥1400px
매우 큰 화면을 위한 브레이크포인트로, 매우 큰 모니터나 고해상도 화면 등을 대응합니다.

■ 미디어 쿼리

● SCSS

// Source mixins


// No media query necessary for xs breakpoint as it's effectively `@media (min-width: 0) { ... }`
@include media-breakpoint-up(sm) { ... }
@include media-breakpoint-up(md) { ... }
@include media-breakpoint-up(lg) { ... }
@include media-breakpoint-up(xl) { ... }
@include media-breakpoint-up(xxl) { ... }

// Usage

// Example: Hide starting at `min-width: 0`, and then show at the `sm` breakpoint
.custom-class {
display: none;
}
@include media-breakpoint-up(sm) {
.custom-class {
display: block;
}
}


● CSS

// X-Small devices (portrait phones, less than 576px)
// No media query for `xs` since this is the default in Bootstrap

// Small devices (landscape phones, 576px and up)
@media (min-width: 576px) { ... }

// Medium devices (tablets, 768px and up)
@media (min-width: 768px) { ... }

// Large devices (desktops, 992px and up)
@media (min-width: 992px) { ... }

// X-Large devices (large desktops, 1200px and up)
@media (min-width: 1200px) { ... }

// XX-Large devices (larger desktops, 1400px and up)
@media (min-width: 1400px) { ... }

2.2 컨테이너 (container)

■ 정의

컨텐츠를 담을 수 있는 너비입니다.
그 안에 내용물을 정렬하고 일정한 간격을 유지하며 브라우저의 너비를 조정합니다.

■ 유형

각 반응형 중단점에 max-width를 설정하는 .container 입니다.

<div class="container"> <!-- Content here --> </div>

정의된 중단점까지 width: 100%를 설정하는 .container-{breakpoint} 입니다.

<div class="container-sm">100% wide until small breakpoint</div>
<div class="container-md">100% wide until medium breakpoint</div>
<div class="container-lg">100% wide until large breakpoint</div>
<div class="container-xl">100% wide until extra large breakpoint</div>
<div class="container-xxl">100% wide until extra extra large breakpoint</div>

전체 너비에 걸쳐 있는 컨테이너에 .container-fluid를 사용합니다.

<div class="container-fluid"> ... </div>

2.3 그리드 (grid)

■ 정의

그리드는 화면 크기에 따라 다른 레이아웃을 쉽게 조절하고, 유연하게 반응형 웹 페이지를 디자인할 수 있도록 도와줍니다.
12개의 열(column)과 행(row)으로 구성되어 있으며, 이를 조합해 콘텐츠를 배치하고 정렬할 수 있습니다.

■ 기본구조

col-*:
기본적인 그리드 클래스로, 모든 화면 크기에 대해 해당 컬럼의 크기를 설정합니다.
여기서 *에는 1부터 12까지의 숫자가 들어갑니다.

col-sm-*, col-md-*, col-lg-*:
특정 화면 크기에 대한 그리드 클래스로, 각각의 클래스는 작은 디바이스(스몰: sm), 중간 크기 디바이스(미디움: md), 큰 디바이스(라지: lg)에 적용됩니다.
각각의 화면 크기에 맞게 컬럼의 크기를 설정할 수 있습니다.

그리드 시스템 클래스
xs (스마트폰용): col- 화면의 너비가 576px 미만
sm(태블릿용): col-sm- 화면의 너비가 576px 이상
md (데스크탑용): col-md- 화면의 너비가 768px 이상
lg (큰 데스크탑용): 화면의 너비가 992px 이상

2.5 거터 (Gutters)

■ 정의

거터는 열 사이의 패딩으로, Bootstrap 그리드 시스템에서 콘텐츠의 간격을 맞추고 정렬하는 데에 사용됩니다.

■ 작동원리

거터의 너비는 1.5rem (24px)에서 시작합니다.
거터는 반응형으로 조정될 수 있습니다.
-0은 열(Column) 사이의 간격을 없애거나 최소화하며, g-5는 가장 큰 간격을 제공합니다.
gx-*, gy-*: 가로(gx)와 세로(gy) 간격을 설정하는 클래스입니다.
gx-*는 열(Column) 사이의 가로 간격을 조절하고, gy-*는 행(Row) 사이의 세로 간격을 조절합니다.


■ 수평 거터 .gx-* 클래스를 사용하여 수평 거터 너비를 제어할 수 있습니다.

<div class="row gx-5">

</div>

2.6 레이아웃용 유틸리티 (Utilities for layout)

■ 유틸리티 소개

Bootstrap에서는 더 빠른 모바일 친화적 반응형 개발을 위해서 콘텐츠 표시, 숨기기, 정렬, 간격 조정을 위한 수십 개의 유틸리티 클래스가 포함되어 있습니다.

* display 속성의 공통값을 반응형으로 전환하려면 디스플레이 유틸리티를 사용합니다.

* 대부분의 컴포넌트는 flexbox가 활성화된 상태로 만들어졌습니다.

* margin 및 padding 간격 유틸리티를 사용하여 요소 및 컴포넌트의 간격과 크기를 제어합니다.

* 가시성 유틸리티를 사용하여 요소의 visibility를 전환할 수 있습니다.


2.7 Z-index

■ 정의

z-index는 Bootstrap 그리드 시스템의 일부는 아니지만 컴포넌트가 서로 중첩되고 상호 작용하는 방식에서 중요한 역할을 담당합니다.
